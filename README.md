# DisableEnd

A simple quick-toggleable fabric mod to disable the end. To toggle the end run `/gamerule doDisableEnd true`

> [!NOTE]
> OwO lib is required from version 0.2-0.6

## Requirements

* Fabric API
* Mod is serverside ONLY and will not run clientside
